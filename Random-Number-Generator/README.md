# Random Number Generator

💁‍♂️ [项目预览](https://hymanchoi.github.io/Treasure-Bag/Random-Number-Generator/index.html)

### 项目介绍

一个随机数生成器

### 相关知识

1. Bootstrap

### 使用说明

1. 左侧输入框输标点号，用一个空格作为分隔。(例如：1 2 3 4 5 6 7)
2. 右侧输入框输抽取个数，要不大于左侧输入框标点数量。
3. 下方输入框输生成组的个数。
4. 点击生成/复制/清空按钮。
